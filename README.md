# 🏪 Warehouse CRM - Система управления складом

**Профессиональная система учёта товаров для малого и среднего бизнеса**

## 📦 Основные возможности
- 📊 Просмотр всех товаров с фильтрацией
- ➕ Добавление новых товаров с указанием цены и количества
- 💰 Продажа товаров с автоматическим расчётом суммы
- 🔍 Поиск самого дорогого/дешёвого товара
- 📈 Аналитика остатков и общей стоимости склада
- 💾 Автоматическое сохранение данных в JSON
- 📄 Экспорт данных в удобном формате

## 🚀 Быстрый старт

### Требования
- Python 3.8 или новее
- Установленный Git (для разработки)
