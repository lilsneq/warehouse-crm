# Система управления складом
	Мой первый git

Программа для учёта товаров на складах компаний.

## Возможности:
- Просмотр всех товаров
- Добавление новых товаров
- Продажа товаров
- Поиск самого дорогого товара
- Аналитика поставщиков

## Как запустить:
1. Установи Python 3.x
2. Скачай файл warehouse_system.py
3. Запусти: `python warehouse_system.py`
